# umi-cnode
### 安装
项目地址：（`git clone`）

```shell
git clone https://github.com/aushion/umi-cnode.git
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

### 启动服务

```
umi dev
```

发布代码
```
umi build
```
